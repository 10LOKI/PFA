# ActTogether - Gamification du Bénévolat

Plateforme de gamification du bénévolat académique. Accumulez des points, gravissez les grades et échangez contre des rewards exclusives.

## Fonctionnalités

### Rôles

- **Étudiant** : Participe aux événements, gagne des points, débloque des rewards
- **Partenaire** : Crée des événements, valide les présences via QR, propose des rewards
- **Admin** : Approuve les partenaires (KYC), gère la plateforme

### Événements

- Création d'événements par les partenaires
- Système de QR code pour check-in valide
- Points garantis basés sur la durée + multiplicateur d'urgence
- Inscription et désinscription des étudiants

### Système de Points

- Points awarded upon successful check-in
- Multiplicateurs pour missions urgentes (1x-3x)
- Grades : Bronze → Argent → Or → Platine → Diamant
- Marketplace de rewards basée sur le grade

### Messages

- Messagerie temps réel entre utilisateurs
- Recherche d'utilisateurs
- Bouton "Message" sur la page événements (participants)
- Real-time avec Pusher

### Authentication

- Inscription avec rôle (student/partner)
- Vérification d'email
- Tableau de bord personnalisé par rôle

## Installation

```bash
# Installer les dépendances
composer install
npm install

# Configuration
cp .env.example .env
# Ajouter les clés Pusher dans .env:
# PUSHER_APP_ID=your_app_id
# PUSHER_KEY=your_key
# PUSHER_SECRET=your_secret
# PUSHER_CLUSTER=your_cluster

# Générer la clé
php artisan key:generate

# migrations (tables déjà créées)
php artisan migrate

# Launcher le serveur
php artisan serve
npm run dev
```

## Structure du projet

```
app/
├── Actions/           # Actions métier (RegisterUser, CheckInStudent, etc.)
├── Http/Controllers/  # Contrôleurs
├── Models/            # Modèles Eloquent
└── Events/            # Events pour broadcasting

database/migrations/  # Tables de la DB

resources/views/
├── messages/         # Views messagerie
├── users/            # Recherche utilisateurs
└── events/           # Views événements
```

## Routes

| Route | Description |
|-------|-------------|
| `/` | Page d'accueil |
| `/dashboard` | Tableau de bord ( selon rôle) |
| `/events` | Liste des événements |
| `/events/{id}` | Détail événement + participants |
| `/messages` | Liste des conversations |
| `/messages/{id}` | Conversation |
| `/users` | Recherche utilisateurs |
| `/rewards` | Marketplace rewards |
| `/profile` | Profil utilisateur |
| `/admin/kyc` | Admin - Approbation partenaires |
